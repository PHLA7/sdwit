# PROJETO SITE SDWIT.COM.BR

## OBJETIVO

Desenvolver um site corporativo para a SDWIT.

A SDWIT é uma consultoria especializada em Tecnologia, Segurança da Informação e Transformação Digital.

O objetivo do site não é vender produtos ou fabricantes, mas posicionar a SDWIT como uma empresa consultiva capaz de ajudar organizações a resolver desafios de tecnologia, segurança e infraestrutura.

O site deve transmitir:

* Credibilidade
* Sofisticação
* Conhecimento técnico
* Capacidade consultiva
* Proximidade com o cliente
* Modernidade

O visual deve ser corporativo e premium.
